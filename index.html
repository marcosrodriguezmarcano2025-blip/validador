Generador de Credenciales QR para Personal Autorizado

Este sistema lee bases de datos de personal médico o administrativo directamente desde un archivo exportado de Google Sheets (en formato CSV). El procesamiento ocurre íntegramente en su computadora. Ningún nombre, ID ni datos confidenciales son transmitidos a internet, garantizando el cumplimiento estricto del secreto clínico y protección de datos internos.

1. Cargar Archivo de Datos

Seleccionar o soltar archivo .CSV Exporte desde Google Sheets como "Valores separados por comas (.csv)"
¿No tiene un archivo listo?

2. Configuración del QR

El sistema unirá esta dirección web con la Cédula (o ID) de cada persona.

Estructura Recomendada de Columnas

Para un mapeo perfecto, asegúrese de que su archivo de Google Sheets tenga las siguientes cabeceras en la primera fila (los acentos no importan):

ID Código interno de empleado
Cédula Documento de identidad nacional
Nombre Nombre completo
Cargo Especialidad o rol
Departamento Área o unidad médica
